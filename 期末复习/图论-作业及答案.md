# 图论综合练习题（修正版）
## T1 完全图相关问题
1. **完全图$K_n$的边数**：完全图$K_n$中任意两个不同结点之间都存在一条边，根据组合数学原理，边数为$\binom{n}{2} = \frac{n(n - 1)}{2}$。
2. **$K_n$是否为欧拉图（一笔画问题）**：欧拉图的判定定理为：一个无向连通图是欧拉图，当且仅当该图中所有结点的度数均为偶数，且图是连通的（完全图$K_n$本身连通）。
    - 当$n = 1$时，$K_1$无边，度数为$0$（偶数），可视为平凡欧拉图；
    - 当$n = 2$时，$K_2$两个结点度数均为$1$（奇数），不是欧拉图；
    - 当$n \geq 3$时，$K_n$中每个结点的度数为$n - 1$：
      - 若$n$为奇数（$n - 1$为偶数），则$K_n$是欧拉图；
      - 若$n$为偶数（$n - 1$为奇数），则$K_n$不是欧拉图。
3. **$K_n$中哈密尔顿回路的数量**：哈密尔顿回路是指经过图中所有结点一次且仅一次，最后回到起点的回路。
    - 对于无向完全图$K_n$（$n \geq 3$），每个哈密尔顿回路可按起点和方向不同重复计数：
      - 固定起点后，剩余$n - 1$个结点的排列数为$(n - 1)!$；
      - 由于回路无方向（顺时针与逆时针视为同一条回路），需除以$2$。
    - 因此，哈密尔顿回路的数量为$\frac{(n - 1)!}{2}$。

## T2 简单图边数的上下限及证明
### 1. 边数范围
- 上限：$0 \leq |E| \leq \binom{n}{2} = \frac{n(n - 1)}{2}$（$n = |V|$为结点数）；
- 下限：$|E| \geq n - 1$（连通简单图），非连通简单图下限为$0$。

### 2. 证明
- 上限证明：简单图中无重边和环，任意两个结点最多连一条边，故边数最多为从$n$个结点中选$2$个的组合数，即$\frac{n(n - 1)}{2}$；边数最少为$0$（空图）。
- 下限证明（连通简单图，基于树的性质）：
  - 树的定义：连通且无环的简单图，其边数$|E| = n - 1$；
  - 若连通简单图$G$的边数$|E| < n - 1$，根据树的性质，任何连通图的边数至少为$n - 1$（树是边数最少的连通图），矛盾；
  - 因此，连通简单图边数下限为$n - 1$。

## T3 连通图的判断方法及连通性判定算法
### 1. 判断方法
- 定义法：图中任意两个不同结点之间都存在至少一条路径；
- 树性质法：若图无环且边数$|E| = n - 1$，则为连通图；
- 邻接矩阵法：通过邻接矩阵的幂运算，若矩阵中所有元素（除对角线外）均非零，则图连通；
- 并查集法：通过合并相邻结点的集合，最终所有结点属于同一集合则图连通。

### 2. 连通性判定算法设计
#### （1）深度优先搜索（DFS）
- 步骤：
  1. 任选一个起点结点，标记为已访问；
  2. 递归访问该结点的所有未访问邻接结点，标记为已访问；
  3. 遍历结束后，若所有结点均被标记为已访问，则图连通，否则不连通。

#### （2）广度优先搜索（BFS）
- 步骤：
  1. 任选一个起点结点，加入队列并标记为已访问；
  2. 出队队首结点，访问其所有未访问邻接结点，加入队列并标记为已访问；
  3. 队列空后，若所有结点均被标记为已访问，则图连通，否则不连通。

#### （3）并查集算法
- 步骤：
  1. 初始化每个结点为独立集合；
  2. 遍历所有边，将边的两个端点所在集合合并；
  3. 遍历结束后，若所有结点属于同一集合，则图连通，否则不连通。

## T4 哈密尔顿图判定定理的证明（Dirac定理）
**定理**：若简单图$G = (V, E)$中，任意不相邻结点$u, v$满足$\deg(u) + \deg(v) \geq n$（$n = |V|$），则$G$为哈密尔顿图。

### 证明（反证法）
假设$G$是满足条件但非哈密尔顿图的最大简单图（即添加任意一条边后变为哈密尔顿图）。
1. 由于$G$非哈密尔顿图，存在最长路径$P = v_1v_2\cdots v_k$（$k \leq n$），且$v_1$与$v_k$不相邻（否则$P$构成回路，可扩展为更长路径，与最长路径矛盾）；
2. 设$S = \{v_i \mid v_1v_i \in E\}$，$T = \{v_i \mid v_{i - 1}v_k \in E\}$，则$S \cap T = \emptyset$（若存在$v_i \in S \cap T$，则$v_1v_2\cdots v_{i - 1}v_kv_{k - 1}\cdots v_i v_1$为哈密尔顿回路，矛盾）；
3. 因此$\deg(v_1) + \deg(v_k) = |S| + |T| \leq |V(P)| - 1 = k - 1 \leq n - 1$，与条件$\deg(u) + \deg(v) \geq n$矛盾；
4. 故假设不成立，$G$为哈密尔顿图。

## T5 哈密尔顿图的边添加等价性证明
**定理**：设$G = (V, E)$为简单图，$u, v \in V$不相邻且$\deg(u) + \deg(v) \geq n$（$n = |V|$），则$G$是哈密尔顿图当且仅当$G' = G \cup \{(u, v)\}$是哈密尔顿图。

### 证明
1. 必要性：若$G$是哈密尔顿图，则$G'$包含$G$的哈密尔顿回路，故$G'$是哈密尔顿图；
2. 充分性：假设$G'$是哈密尔顿图但$G$不是，则$G'$的所有哈密尔顿回路都必须包含边$(u, v)$；
   - 设$C$是$G'$的哈密尔顿回路，即$C = u v v_3 v_4 \cdots v_n u$，去掉边$(u, v)$后得到路径$P = v v_3 v_4 \cdots v_n u$；
   - 设$\deg_G(u) = a$，$\deg_G(v) = b$，则$\deg_{G'}(u) = a + 1$，$\deg_{G'}(v) = b + 1$；
   - 类似T4的证明，可得$\deg_G(u) + \deg_G(v) = a + b \leq n - 1$，与条件$a + b \geq n$矛盾；
   - 故$G$必为哈密尔顿图。

## T6 图论中基本的证明方法
1. **直接证明法**：根据定义、公理或已知定理，直接推导结论（如证明树的边数为$n - 1$）；
2. **反证法**：假设结论不成立，推出与已知条件或定理矛盾的结果（如T4、T5的证明）；
3. **数学归纳法**：对结点数、边数或图的阶数归纳，基础步验证小范围成立，归纳步假设$k$阶图成立，证明$k + 1$阶图成立（如T9树的边数证明）；
4. **构造法**：直接构造满足条件的图、路径或回路（如哈密尔顿回路、欧拉图的构造）；
5. **归纳法结合反证法**：综合两种方法的优势，适用于复杂定理证明（如平面图相关定理）；
6. **抽屉原理**：利用“有限个抽屉放无限个元素，必有一个抽屉含多个元素”，证明存在性（如证明平面图中存在度数≤5的结点）。

## T7 树的基本性质及平面图相关结论
### 1. 树的定义
连通且无环的简单图称为树，记为$T = (V, E)$。

### 2. 树的基本性质及相互证明
- 性质1：树的边数$|E| = n - 1$（$n = |V|$）；
- 性质2：树是连通的，且去掉任意一条边后变为非连通图；
- 性质3：树无环，且添加任意一条边后恰好形成一个环；
- 性质4：树中任意两个结点之间存在唯一一条路径。

### 相互证明（以性质1⇨性质2为例）
- 性质1⇒性质2：若$T$连通且$|E| = n - 1$，假设去掉某条边$e$后仍连通，则$T - e$是连通图且边数为$n - 2$，与“连通图边数≥$n - 1$”矛盾，故去掉任意边后非连通；
- 性质2⇒性质1：若$T$连通且去任意边非连通，则$T$无环（否则去掉环上一条边仍连通），故$T$是树，边数为$n - 1$。

### 3. 树的平面图性质
- 树是平面图：树无环，满足平面图“无交叉边”的定义，可嵌入平面；
- 结点数、边数、面数的关系：根据欧拉公式$n - e + f = 2$，代入$e = n - 1$，得$f = 2 - n + (n - 1) = 1$，即树只有一个面（外部面）。

## T8 平面图欧拉定理的证明
**欧拉定理**：设连通平面图$G = (V, E, F)$，其中$n = |V|$，$e = |E|$，$f = |F|$（含外部面），则$n - e + f = 2$。

### 证明（数学归纳法）
1. 基础步：当$e = 0$时，$G$是孤立结点图，$n = 1$，$f = 1$，则$1 - 0 + 1 = 2$，成立；
2. 归纳步：假设对所有边数为$k$的连通平面图，欧拉公式成立。现考虑边数为$k + 1$的连通平面图$G$：
   - 若$G$无环（树），则$e = n - 1$，$f = 1$，代入得$n - (n - 1) + 1 = 2$，成立；
   - 若$G$有环，任取环上一条边$e$，则$G' = G - e$仍是连通平面图，边数为$k$，面数为$f - 1$（去掉边$e$后两个相邻面合并）；
   - 由归纳假设，$n - k + (f - 1) = 2$，即$n - (k + 1) + f = 2$，故边数为$k + 1$时公式成立；
3. 由归纳法，对所有连通平面图，欧拉公式成立。

### 非连通平面图的推广
若$G$是有$w$个连通分量的平面图，则欧拉公式为$n - e + f = w + 1$。

## T9 树的结点数与边数关系的数学归纳法证明
**定理**：设树$T = (V, E)$，$n = |V|$，则$|E| = n - 1$。

### 证明
1. 基础步：当$n = 1$时，$T$是孤立结点，$|E| = 0 = 1 - 1$，成立；
2. 归纳步：假设对所有结点数为$k$的树，边数为$k - 1$。现考虑结点数为$k + 1$的树$T$：
   - 由于$T$连通且无环，任取一个叶子结点$v$（度数为$1$的结点）；
   - 去掉$v$及其关联的边$e$，得到图$T' = T - v - e$，$T'$仍是树（无环且连通），结点数为$k$；
   - 由归纳假设，$T'$的边数为$k - 1$，故$T$的边数为$(k - 1) + 1 = k = (k + 1) - 1$；
3. 由归纳法，对所有树，$|E| = n - 1$成立。

## T10 图论中的基本数量关系及平面图相关证明
### 1. 基本数量关系
- 握手定理：无向图中所有结点的度数之和等于边数的$2$倍，即$\sum_{v \in V} \deg(v) = 2e$；
- 树的边数：$e = n - 1$；
- 平面图欧拉公式：连通平面图$n - e + f = 2$；
- 平面图边数上限：连通简单平面图（$n \geq 3$）满足$e \leq 3n - 6$；含三角形的连通简单平面图满足$e \leq 2n - 4$。

### 2. 证明“平面图中至少存在一个结点的度数≤5”
**证明（反证法）**：
假设连通简单平面图$G$中所有结点的度数$\deg(v) \geq 6$，则由握手定理：
$\sum_{v \in V} \deg(v) = 2e \geq 6n$，即$e \geq 3n$；
但连通简单平面图（$n \geq 3$）的边数上限为$e \leq 3n - 6$，矛盾；
因此，平面图中至少存在一个结点的度数≤5。

## T11 该命题对五色定理证明的意义
五色定理：任何平面图都可以用5种颜色着色，使得相邻结点颜色不同。

### 意义
“平面图中至少存在一个结点的度数≤5”是五色定理证明的关键基础：
1. 五色定理的证明采用数学归纳法，对平面图的结点数$n$归纳；
2. 当$n \leq 5$时，定理显然成立；
3. 假设对所有$n = k$的平面图可5着色，考虑$n = k + 1$的平面图$G$：
   - 由上述命题，$G$中存在结点$v$满足$\deg(v) \leq 5$；
   - 去掉$v$得到平面图$G - v$，由归纳假设$G - v$可5着色；
   - 由于$v$的邻结点数≤5，最多占用5种颜色，因此可选择一种未被邻结点使用的颜色给$v$着色，使得$G$可5着色；
4. 若无此命题，无法保证归纳步中存在度数较小的结点，导致着色步骤无法推进。

## T12 基础问题示例（修正版）
1. 设$G$是一个具有8个结点、12条边的无向简单图。计算$G$中所有结点的度数之和。
   - 解：由握手定理$\sum_{v \in V} \deg(v) = 2e$，代入$e = 12$，得度数之和为$2 \times 12 = 24$。

2. 设$G$是一个具有6个结点的无向简单图，且每个结点的度数均不小于3。判断$G$是否一定连通？是否一定是哈密尔顿图？分别说明理由。
   - 解：
     - 一定连通：假设$G$不连通，设其连通分量为$G_1, G_2, \cdots, G_k$（$k \geq 2$），则至少有一个连通分量$G_i$满足$|V(G_i)| \leq 3$。该分量中任意结点的最大度数为$|V(G_i)| - 1 \leq 2$，与“每个结点度数≥3”矛盾，故$G$连通；
     - 一定是哈密尔顿图：对任意两个不相邻结点$u, v$，$\deg(u) + \deg(v) \geq 3 + 3 = 6 = n$，满足Dirac定理条件，故$G$为哈密尔顿图。

3. 设$G$是一个连通的简单平面图，有10个结点、15条边。求$G$的面数（包括外部面）。
   - 解：由连通平面图欧拉公式$n - e + f = 2$，代入$n = 10$，$e = 15$，得$f = 2 - n + e = 2 - 10 + 15 = 7$，故面数为7。

4. 完全图$K_5$是否是欧拉图？是否是哈密尔顿图？分别说明理由。
   - 解：
     - 是欧拉图：$K_5$连通，且每个结点的度数为$5 - 1 = 4$（偶数），满足欧拉图判定定理；
     - 是哈密尔顿图：所有$n \geq 3$的完全图中，任意两个结点之间都有边，故存在经过所有结点的回路，即哈密尔顿回路，因此$K_5$是哈密尔顿图。

5. （i）一棵有7个结点的生成树有多少条边？（ii）根据凯莱定理，7个带标号结点可以构成多少棵不同的生成树？
   - 解：
     - （i）生成树是连通且无环的子图，边数为$n - 1$，代入$n = 7$，得边数为$7 - 1 = 6$；
     - （ii）凯莱定理指出，$n$个带标号结点的生成树数量为$n^{n - 2}$，代入$n = 7$，得数量为$7^{7 - 2} = 7^5 = 16807$棵。
